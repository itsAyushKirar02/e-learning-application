# This is a web based Application 

<p>
An E-Learning Platform is a web-based application designed to provide online education and learning resources. It enables students to access courses, study material, and track their learning progress, while allowing instructors/admins to manage courses and content efficiently.
</p>

<ol>
  <li><strong>Student Features</strong>
    <ul>
      <li>User registration and login</li>
      <li>Browse courses</li>
      <li>Enroll in courses</li>
      <li>Access learning materials</li>
    </ul>
  </li>

  <li><strong>Instructor / Admin Features</strong>
    <ul>
      <li>Create and manage courses</li>
      <li>Upload video lectures and materials</li>
      <li>Manage students and enrollments</li>
    </ul>
  </li>

  <li><strong>Authentication</strong>
    <ul>
      <li>Secure login and signup</li>
      <li>Role-based access (Student / Admin)</li>
    </ul>
  </li>
</ol>
